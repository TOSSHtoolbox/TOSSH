TOSSH: A Toolbox for Streamflow Signatures in Hydrology

This documentation gives a technical overview of TOSSH: A Toolbox for Streamflow Signatures in Hydrology (Gnann et al., 2021). TOSSH is a Matlab toolbox that provides accessible, standardised signature calculations, with clear information on methodological decisions and recommended parameter values.
