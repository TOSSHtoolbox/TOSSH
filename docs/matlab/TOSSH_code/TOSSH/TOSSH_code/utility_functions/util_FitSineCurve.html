util_FitSineCurve

PURPOSE

util_FitSineCurve fits sine curve to time series.

SYNOPSIS

function [A, phi, k] = util_FitSineCurve(x, y, w)

DESCRIPTION 

util_FitSineCurve fits sine curve to time series.
   y = A*sin(w*x + phi) + k;

   INPUT
   x: independent variable (typically time)
   y: dependent variable (typically flow)
   w: angular frequency

   OUTPUT
   A: amplitude
   phi: phase
   k: offset

   EXAMPLE
   x = [0:5*365]';
   w = 2*pi/365;
   y = 1 + sin(w.*x + pi/2);
   [A,phi,k] = util_FitSineCurve(x,y,w)
