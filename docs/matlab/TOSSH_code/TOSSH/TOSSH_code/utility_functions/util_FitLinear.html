util_FitLinear

PURPOSE

util_FitLinear fits linear function and returns parameters and residuals.

SYNOPSIS

function [a, b, R2] = util_FitLinear(x,y)

DESCRIPTION 

util_FitLinear fits linear function and returns parameters and residuals.
   y = a + b*x;
   
   INPUT
   x: independent variable
   y: dependent variable

   OUTPUT
   a: offset parameter
   b: slope parameter
   R2: residuals
   
   EXAMPLE
   x = [0:5];
   y = 1 + 2*x;
   [a, b, R2] = util_FitLinear(x,y)
