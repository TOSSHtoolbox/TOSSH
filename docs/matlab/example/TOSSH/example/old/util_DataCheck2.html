util_DataCheck2

PURPOSE

util_DataCheck Checks data for various things (e.g. unrealistic values).

SYNOPSIS

function [error_flag, error_str, timestep, t] = util_DataCheck2(Q, t, varargin)

DESCRIPTION 

util_DataCheck Checks data for various things (e.g. unrealistic values).

   INPUT
   Q: streamflow [mm/timestep]
   t: time [Matlab datetime]
   OPTIONAL
   P: precipitation [mm/timestep]
   PET: potential evapotranspiration [mm/timestep]
   T: temperature [°C]

   OUTPUT
   error_flag: 0 (no error), 1 (warning), 2 (error)
   error_str:
   timestep: (median) timestep of data
   t: time in Matlab datetime format

   EXAMPLE
   % load example data
   data = load('./example/example_data/33029_daily.mat');
   Q = data.Q;
   t = data.t;
   P = data.P;
   dataCheck = util_DataCheck(Q, t, 'P', P);
